HTML

1. Waht is HTML?

    HTML stands for Hyper Text Markup Language which is used for creating web pages and web applications released in 1993. HTML describes the structure of a Web page and HTML tell the browser what to display and how to display the content in Web Pages.

2. HTML5 new features?

    Some new features of HTML5 :

  1. It supports multimedia like video and audio.
  2. Added Header, Footer, Nav, and Progress tags.
  3. Javascript capable to run in the background.
  4. Added Canvas tag newly it is used to draw the images in different shapes.
  5. It supports storing offline data using SQL databases and in the application cache.

3. Difference between <p> and <span> tags?

   <p> is a block-level tag for paragraphs. <span> is a line-level tag that in and of itself doesn't do anything, but is useful for defining a specific style to a string. <span> can be used within <p>, but <p> cannot be used within a <span>.

4. What is hoisting in javascript?

  Hoisting is a JavaScript default behavior where variables and function declarations are moved to the top of their scope before code execution. In simple words a variable or function can be used before it has been declared. 

  
  Error
  console.log(number); // undefined
  var number = 10;

  Correct
  var number = 10;
  console.log(number); // 10

5. Difference between <section> and <div> tags?

   The <section> tag defines sections in a document, such as chapters, headers, footers, or any other sections of the html document.

   The <div> tag defines a division or a section in an HTML document. The tag is used to group block elements to format / styling them with CSS.
